## ⚙️ Installation

Play through app site, no installation required. Otherwise clone into local machine and open on IDE:

```bash
# clone the repo
git clone https://github.com/sifzerda/nx-minesweeper.git

# move into directory
cd nx-minesweeper

# install dependencies
npm install

# run server
npm run start
```
